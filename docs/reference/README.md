# AoE2 Referans Dokümantasyon

> Bu klasör Age of Empires II'nin **gerçek mekaniklerini** sayısal detayla saklar.
> Oyun tasarım kararları için tek kaynak olarak kullanılır.
>
> **Fark:** `docs/01-12.md` dosyaları "AoA'da ne var/ne eksik?" sorusunu yanıtlar.

> **Versiyon notu:** Ağırlıklı olarak Age of Empires II: Definitive Edition (2019) baz alınmıştır.
> Orijinal The Age of Kings (1999) ile bazı stat farkları olabilir.
